# Flask Web App

## To Setup:
Verify you have the latest version of Python Installed.

```bash
git clone https://github.com/slittl46/flask-web-notes-app.git
```

```bash
pip install -r required.txt
```

## To Run the App

```bash
python main.py
```

## To View the App in Browser

Go to 'http://127.0.0.1:5000'
